# Pricing

## Free Tier

{% hint style="success" %}
GooseCode is a **free first** tool and the core features will always be free.
{% endhint %}

## Premium Tier - Planned

{% hint style="warning" %}
We're working on a premium tier which provides advanced functionality on top of the free tier model
{% endhint %}

### Individual Contributor

The individual contributor plan is designed for super-users who need advanced features but do not belong to a team.&#x20;

Expected Features:

* MCP tooling to generate flows using AI from your IDE
* Integrations with third-party platforms such as Linear and Google Drive
* Automated GitHub project generation
* Charting visualizations for CSVs
* Secure exported files that can demonstrate a complex code flow for documentation
* Shareable links
* Pages within a canvas
* Demonstration mode

### Team

The team plan is designed for small teams such as startups with all the functionality of Individual Contributor but with more seats.

Expected Features:

* Shareable folders
* Encrypted e2e sending for transferring large files and secrets between team members

### Business

The business plan is designed for large organizations that require extended support, security and authentication functionality.

Expected Features:

* SAML authentication
* MFA
* Spaces for different business units
